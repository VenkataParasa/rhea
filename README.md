# RHEC Redesign Demo

A Next.js App Router design demo built from the RHEC requirements. Content is local mock data; forms and third-party actions are deliberately non-submitting placeholders.

## Run locally

```bash
npm install
npm run dev
```

## Structure

- `app/` — independently navigable public pages and dynamic detail templates
- `components/` — shared header, footer, cards, CTAs, and page hero components
- `data/` — typed local mock data for programs, announcements, events, documents, and staff
